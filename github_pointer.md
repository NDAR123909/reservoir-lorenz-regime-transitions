# GitHub

This repository is laid out per the project charter (section 8) and initialized
with git locally. To publish it, create an empty repository on GitHub and point
this one at it:

```bash
git remote add origin https://github.com/<username>/reservoir-lorenz-regime-transitions.git
git branch -M main
git push -u origin main
```

Once pushed, replace this file's contents with the canonical URL so later
sessions and the paper can reference it.
